# 🚀 KJs TRD Trading Terminal

A **Personal Trading Operating System** built with Python, HTML, CSS, and JavaScript.

---

## ✨ Features

- ✅ **Professional Trading Terminal UI**
- ✅ **Interactive Candlestick Chart** (Zoom, Pan, Hover)
- ✅ **Multiple Timeframes** (1m, 5m, 15m, 1h, 1d)
- ✅ **Indicators** (EMA, VWAP, RSI, MACD)
- ✅ **Python Studio** for creating strategies
- ✅ **Module Library** for managing strategies
- ✅ **Backtesting Engine** for historical testing
- ✅ **Paper Trading** for risk-free practice
- ✅ **Auto Trading** with Dhan/Zerodha brokers
- ✅ **Real Market Data** via Dhan API
- ✅ **Notifications** (Telegram, WhatsApp, Email)
- ✅ **Dark Theme** for professional look
